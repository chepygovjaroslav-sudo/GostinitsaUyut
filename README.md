# Гостиница «Уют» — Информационная система учёта услуг

## Описание проекта
Система предназначена для автоматизации учёта проживания и дополнительных услуг в малой гостинице.

## Основные функции
- Бронирование номеров
- Заселение и выселение гостей
- Учёт дополнительных услуг (питание, трансфер, прачечная)
- Формирование счетов
- Отчёты по загрузке и доходам

## Технологии
- C# / Windows Forms
- MS Access (база данных)
- Git (контроль версий)
